# Phase 3 Demo Flow

1. Sign in as a worker and complete onboarding.
2. Show dynamic weekly premium with live grid conditions and ISS impact.
3. Open the worker dashboard:
   - live zone watch
   - predictive alert
   - earnings protected
   - Telegram link state
4. Open admin overview:
   - loss ratio
   - payout summary
   - predictive hotspots
5. Open admin disruption map and trigger one live/manual disruption on an insured grid.
6. Show the worker dashboard refreshing with:
   - disruption banner
   - claim card
   - payout or review state
   - Telegram notification
7. If claim is flagged, approve it from admin fraud/claims review.
8. Show approved-after-review payout receipt and updated payout timeline.
